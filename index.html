Relação entre Índices Pluviométricos e Criminalidade na Capital de São Paulo

Como usar

Para que um gráfico seja gerado, é necessário um arquivo Excel com os dados criminais (OcorrenciaMensal(Criminal)-101 DP - Jardim das Imbuias_20240522_191258.xlsx) e um arquivo CSV com os dados climáticos (INMET_SE_SP_A771_SAO PAULO - INTERLAGOS_01-01-2023_A_31-12-2023.CSV). Após o upload dos arquivos, selecione a coluna do arquivo Excel e a linha do arquivo CSV que deseja comparar. Por fim, clique em "Gerar Gráfico" para visualizar a relação entre os dados.

CSV

  • Os nomes das colunas devem estar na linha 9
  • Os dados devem estar após a linha 9
  • A primeira coluna deve ser a data
  • A terceira coluna deve ser a quantidade de chuva
  • O delimitador é ";"

Excel

  • A primeira linha deve ser a linha com os meses
  • Cada coluna da primeira linha representa um mês
  • A primeira coluna deve ser o tipo de crime
  • Cada coluna da linha representa o total daquele tipo de crime naquele mês

Para que mais dados possam ser adquiridos, acesse https://portal.inmet.gov.br/dadoshistoricos para o arquivo CSV dos dados climáticos e https://www.ssp.sp.gov.br/estatistica/dados-mensais para o arquivo Excel de dados criminais.

LGPD

Implementamos os padrões ISO 27001 para garantir a integridade e precisão dos dados. Nossos dados são anonimizados e não incluem dados sensíveis sobre indivíduos nos dados utilizados, assegurando a proteção das informações coletadas.
