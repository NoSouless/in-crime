Como usar

Para que um gráfico seja gerado, é necessário um arquivo Excel com os dados criminais (OcorrenciaMensal(Criminal)-101 DP - Jardim das Imbuias_20240522_191258.xlsx) e um arquivo CSV com os dados climáticos (INMET_SE_SP_A771_SAO PAULO - INTERLAGOS_01-01-2023_A_31-12-2023.CSV). Após o upload dos arquivos, selecione a coluna do arquivo Excel e a linha do arquivo CSV que deseja comparar. Por fim, clique em "Gerar Gráfico" para visualizar a relação entre os dados. Originalmente, usamos os dados de 2023, mas você pode escolher qualquer ano.

LGPD

Implementamos os padrões ISO 27001 para garantir a integridade e precisão dos dados. Nossos dados são anonimizados e não incluem dados sensíveis sobre indivíduos nos dados utilizados, assegurando a proteção das informações coletadas.
